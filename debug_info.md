# デバッグ・作業修正ログ (2026-03-18 14:45)

## 1. Google CSV 自動認識の修正
- **事象**: Google連絡先CSVをドラッグ＆ドロップしても機種が自動で切り替わらない。
- **原因**: 以前の判別ロジックが「列数の一致」を前提としていたため、Google形式（可変列）がスキップされていた。
- **修正**: `src/js/services/csv-parser.js` を修正し、ヘッダーに "First Name" 等のシグネチャが含まれている場合に優先的に「Google連絡先」として認識するように変更。

## 2. GitHub の同期不整合の解消
- **状況**: ローカルの `dev` ブランチは最新だが、GitHub上の `main` や `master` が古いままだった可能性がある。
- **対処**: 強制プッシュにより、`dev` ブランチの内容を `main` / `master` に完全に同期させた。

## 3. その他の修正内容（再確認）
- **名前の統合**: First Nameが空でも統合され、前後のスペースが除去されるよう改善。
- **電話番号の分割**: 全スロットを一旦リスト化してから順番に割り当てることで、`::: ` 連結問題を確実に解消。
- **半角変換**: ひらがなをカタカナ経由で半角にする処理を `toHalfWidth` に追加。
